## 什么是 Cursor

Cursor 是一款集成了 AI 能力的代码编辑器，基于 VS Code 构建，支持 AI 对话、智能代码补全、自动重构等功能，大幅提升开发效率。

## 安装与设置

1. 访问 [cursor.com](https://cursor.com) 下载安装
2. 可以直接导入 VS Code 的配置和插件
3. 登录账号后即可使用 AI 功能

## 核心功能

### 1. AI 对话（Cmd/Ctrl + L）

在侧边栏打开 AI 对话面板，你可以：

- 询问代码相关问题
- 让 AI 解释某段代码
- 请求生成新功能代码
- 使用 `@` 引用文件、文件夹作为上下文

### 2. 内联编辑（Cmd/Ctrl + K）

选中代码后按快捷键，直接在编辑器中告诉 AI 你想要的修改：

```
"把这个函数改成 async/await 写法"
"添加错误处理"
"优化性能"
```

### 3. Tab 智能补全

Cursor 会根据上下文自动提供代码建议，按 Tab 即可接受。它能理解你的整个项目结构。

## 使用技巧

- **提供清晰的上下文**：引用相关文件，AI 就能给出更准确的建议
- **分步骤操作**：复杂的重构拆分成小步骤，效果更好
- **Review AI 的输出**：始终检查 AI 生成的代码是否正确

## 适用场景

- 快速搭建项目脚手架
- 编写重复性代码
- 学习新语言或框架
- Debug 和代码审查
