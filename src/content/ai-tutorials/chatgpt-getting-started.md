## 什么是 ChatGPT

ChatGPT 是由 OpenAI 开发的大语言模型，能够理解和生成自然语言文本。你可以用它来回答问题、撰写文章、编写代码、翻译语言等。

## 快速开始

1. 访问 [chat.openai.com](https://chat.openai.com) 并注册账号
2. 登录后即可在对话框中输入你的问题
3. ChatGPT 会实时生成回复

## 提示词（Prompt）基础技巧

好的提示词是获得高质量回复的关键：

### 1. 明确你的需求

```
❌ 写一篇文章
✅ 写一篇 800 字的科普文章，主题是量子计算的基本原理，面向没有物理学背景的读者
```

### 2. 指定输出格式

```
请用以下格式回答：
- 一句话总结
- 详细解释（3-5 段）
- 关键要点列表
```

### 3. 提供角色设定

```
你是一位资深的前端开发工程师，请帮我审查以下代码并提出改进建议。
```

## 常见使用场景

- **写作辅助**：起草邮件、文章、报告
- **编程助手**：调试代码、解释概念、生成代码片段
- **学习工具**：解释复杂概念、制定学习计划
- **翻译**：多语言翻译和润色
- **头脑风暴**：生成创意和方案

## 注意事项

> ChatGPT 的回复可能包含错误信息，建议对重要内容进行事实核查。
