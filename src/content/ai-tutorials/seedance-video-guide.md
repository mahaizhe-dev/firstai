## 什么是 Seedance 2.0

Seedance 2.0 是字节跳动于 2026 年 2 月发布的新一代 AI 视频生成模型，被称为「导演级」视频生成工具。GitHub 开源后迅速引发全网关注，豆包 App 一度排队 9 万人。

与上一代相比，Seedance 2.0 的核心突破在于：

- **2K 电影级画质**，分辨率从 1080p 提升至 2K
- **多模态混合输入**，同时接受文字 + 图片 + 视频 + 音频
- **原生音画同步**，双声道立体声，支持环境音效和多语言口型同步
- **导演级可控性**，支持运镜指令、光影控制、视频延长和编辑

## 3 步快速出片

### 第一步：上传素材

打开即梦网页版，进入「视频生成」。上传 1 张高清图片作为首帧参考（建议 1024×1024 以上，无水印）。

### 第二步：编写提示词

好的提示词决定 80% 的效果，推荐使用以下公式：

```
主体 + 动作 + 场景 + 运镜 + 风格 + 画质
```

**示例提示词：**

```
动漫少女在樱花树下慢走，微风拂发，微笑看镜头，
黄昏暖光，电影感，稳定跟拍，2K高清
```

```
一只橘猫趴在窗台上打哈欠，阳光从百叶窗洒入，
浅景深特写，温暖色调，胶片质感
```

```
赛博朋克城市街道，霓虹灯倒映在湿漉漉的地面，
镜头缓缓前推，蒸汽从井盖冒出，电影宽银幕
```

### 第三步：设置参数并生成

| 参数 | 推荐设置 |
|------|----------|
| 画面比例 | 16:9（横屏）/ 9:16（竖屏）/ 1:1（方形） |
| 视频时长 | 新手选 5-10 秒，进阶选 10-15 秒 |
| 分辨率 | 1080P 或 2K |

点击生成，等待 30-90 秒即可预览。

## 进阶：全能参考模式

Seedance 2.0 最强大的能力是**多模态混合输入**。你可以同时上传最多 **9 张图片 + 3 段视频 + 3 段音频**，用 `@` 语法告诉模型每个素材的用途：

```
@Image1 作为角色参考
@Image2 作为场景参考
@Video1 作为运镜参考
@Audio1 作为背景音乐

角色在场景中缓步行走，按照视频运镜方式拍摄，
配合背景音乐节奏
```

### 实用场景

**产品宣传片**：上传产品图 + 品牌 LOGO + 一段节奏感强的 BGM，生成 15 秒产品展示视频。

**短剧分镜**：上传多张角色设定图 + 场景图，分段生成并拼接，制作完整短剧。

**音乐 MV**：上传一段音乐 + 风格参考图，模型会根据音乐节奏自动匹配画面节奏。

## 运镜指令速查

在提示词中加入运镜关键词，可以精确控制摄像机：

| 运镜 | 关键词 | 效果 |
|------|--------|------|
| 推镜 | 镜头前推、zoom in | 从远到近，聚焦主体 |
| 拉镜 | 镜头后拉、zoom out | 从近到远，展示全景 |
| 横移 | 镜头左移/右移、横向跟拍 | 水平方向跟随 |
| 环绕 | 环绕拍摄、360度旋转 | 绕主体旋转 |
| 升降 | 镜头上升/下降、鸟瞰 | 垂直运动 |
| 稳定跟拍 | 稳定跟拍、平滑移动 | 手持跟拍效果 |
| 固定机位 | 固定镜头、静止 | 摄像机不动 |

## 提示词优化技巧

1. **越具体越好**：不写"一个人在走路"，写"穿灰色风衣的女性在雨中撑伞慢走"
2. **加光线描述**：黄昏暖光、正午顶光、霓虹灯光、逆光剪影
3. **加风格关键词**：电影感、胶片质感、动漫风、赛博朋克、水墨风
4. **加画质后缀**：2K高清、超高清、高细节、景深效果
5. **分段生成再拼接**：复杂场景拆成多个 5 秒片段，效果更稳定
