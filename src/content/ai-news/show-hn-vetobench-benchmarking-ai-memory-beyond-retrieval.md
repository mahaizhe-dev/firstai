VetoBench is an open-source benchmark that measures whether AI agents repeat engineering decisions a team has already rejected. Instead of evaluating retrieval accuracy, it evaluates decision quality: does the agent propose a previously vetoed approach, and does it recognize when a memory is no longer valid?
