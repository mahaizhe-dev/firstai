Hey Guys, I would like to showcase Tiny Agents, which is an entirely Rust-based RLM. RLMs are a recent and fantastic new innovation on how LLMs can be reimagined by designing the LLM agent itself to define how it will orchestrate or create its own sub-agent.I also could not find any good LLM harness workflow in Rust, so I built this one, taking massive inspiration from LangGraph and LangChain but also incorporating a Rust-based transpiler to build on-demand LLM graphs all in Rust following the RLM idea.This is the harness that we are currently using in production in openhuman.
