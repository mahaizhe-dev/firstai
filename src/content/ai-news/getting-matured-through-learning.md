I created a product. Since i started this project the only thing that bothered me was, things were not working properly. There was a gap in actually thinking about some feature to introduce in my project and how i implemented those in my project. So, i tried my best to understand about UI/UX, what should be the main focus of my product and how can i build new things quickly. As a new guy i got my hands dirty on various Ai builder tools but as i progressed further i found out there were many hurdles like integration of payment gateways, Email integration, Compliance, Designing Privacy Policy, Documentation, Adding features, Scaling the project. I some how manged to first understand all these and then started implementing them in my project accordingly. Then i came to know that while doing all these things i have grown my skills in different aspects of building a project from the ground-up. I have realized that this is just not about creating a product it is about understanding how things work and how to get the things done according to my project.
