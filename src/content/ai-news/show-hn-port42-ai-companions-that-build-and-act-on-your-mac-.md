Port42 is a native SwiftUI app where AI companions live alongside you in conversations. They build interactive surfaces and act on your system.Two surfaces, one API:Ports are live HTML/CSS/JS apps that companions create mid-conversation. They render inline, pop out into floating windows, dock, resize, persist across sessions. A companion can build you a dashboard, a file browser, a monitoring tool, and it runs right there.Tools let companions act on your machine directly. Clipboard, screen capture, terminal, filesystem, headless browser, audio TTS, AppleScript/JXA automation. The model decides which tools to call and chains them together. "Run my tests." "Take a screenshot and describe what you see." "Read my clipboard, translate it, copy it back."Both surfaces hit the same bridge API. Ports are persistent and interactive. Tools are request/response. We bridge both because computing is still stuck in windows, but conversation is where intent lives.Requires a Claude subscription (via Claude Code OAuth) or Anthropic API key. Permission prompts per companion, per conversation. E2E encrypted (AES-256-GCM). No cloud dependency. Open source (MIT).9 days old, rough edges everywhere, but it works, now at v0.5.0. with 40+ releases.
