Toolport is a local MCP gateway with a native desktop app. Add and authenticate a server once, then connect it to any of 22 supported clients. Secrets stay in the OS keychain.Instead of exposing the full catalog, Toolport gives agents four small meta-tools and searches for the real tools on demand. With 62 tools, that reduced definition overhead from about 24k tokens to 900. In benchmarked tasks, total token use fell by up to 91% at the same success rate.It is MIT licensed with Windows, macOS, and Linux installers.
