I do a fair amount of embedded and hardware work on my mac. The serial tool was always a bother for me. I usually ended up testing on Windows. I built this tool focused on day-to-day bringup, loop back tests, logging and repeatable send flows. it uses Tauri and Rust for speed and a low memory footprint. It is free, open source, MIT license. I have used this on a Mac and built and tested the Linux version. Did not get to building the Windows version yet. Will probably look at it next weekend.
