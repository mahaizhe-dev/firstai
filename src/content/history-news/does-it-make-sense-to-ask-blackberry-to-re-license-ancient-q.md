The 17-year-old sources of QNX (to be found at github.com/vocho/openqnx) don't have a clearly-defined license file/status. In theory, one can use them, experiment with them, but they're neither free-, nor completely open-source.Does it make sense to ask QSS/Blackberry to re-license them under e.g. Apache 2.0 license -- the same license they use for their startup code sources?If yes, does it make sense to write/publish an open petition?
